Dessin aléatoire (et souvent moche)

Niveau : facile
Stylo écrit
Répéter jusqu'à 1=0 (en gros indéfiniment)
 Avancer de RanInt#(1;7)
 Tourner de RanInt#(1;360) degrés
 Si x › 90 Alors
  Aller à x = RanInt#(-90;90)
  y = RanInt#(-20;20)
 Fin
 Si x ‹ -90 Alors
  Aller à x = RanInt#(-90;90)
  y = RanInt#(-20;20)
 Fin
 Si y › 20 Alors
  Aller à x = RanInt#(-90;90)
  y = RanInt#(-20;20)
 Fin
 Si y ‹ -20 Alors
  Aller à x = RanInt#(-90;90)
  y = RanInt#(-20;20)
 Mettre var A = RanInt#(0;1)
 Si A = 0 Alors
  Stylo écrit
 Fin
 Si A = 1 Alors
  Stylo relevé
